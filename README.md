# Modular Scale Viewer
Simple Demo app on Vue.js

Modular Scale Viewer is simple app to customize modular font scale for your design 


## Download
1. Download repo
 `git clone git@github.com:Malahov-web/modular-scale-viewer.git`

## Development
1. Install project
`npm i`

2.  Run live dev
`npm run serve`   

3.  Build prod files 
 `npm run build`
